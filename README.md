# MeowFacts API Demo

A simple web app that displays random cat facts using the MeowFacts API, with a fun animated background of floating cats.

## Files
- index.html – Main HTML structure  
- meow.css – Styling and floating cat animation  
- meow.js – JavaScript for fetching cat facts from the API  

## Features
- Fetches random cat facts from an API  
- Button to load a new fact  
- Animated floating cats in the background  
- Clean and simple UI  

## How to Use
1. Download or clone the project  
2. Make sure all files are in the same folder  
3. Open index.html in your browser  
4. Click Get New Fact  

## API Used
- https://meowfacts.herokuapp.com/
